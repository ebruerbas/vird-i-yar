# Bildirim Sesi

`tools/make-ezan-sound.sh` script'i bu klasöre iki dosya üretir:

- `ezan_kisa.caf` — iOS bildirim sesi (28 sn). Xcode'da App hedefine eklenmelidir.
- `ezan_kisa.wav` — Android bildirim sesi. `android/app/src/main/res/raw/` altına kopyalanmalıdır.

Ses kaynağı: Nasır el-Katami ezanı, [archive.org/details/1azan](https://archive.org/details/1azan) (CC BY-NC 3.0).
